# My City Weather

A simple weather web app that shows the current weather for any city using the OpenWeather API.

Live demo: https://pankaj-teron.github.io/Weather-App/

---

## Features

- Search weather by city name
- Displays:
  - Current temperature (Â°C)
  - City name
  - Humidity
  - Wind speed
  - Weather icon (clear, clouds, rain, etc.)
- Shows a **default city** on page load
- Basic error handling for invalid city names

---

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **OpenWeather API**
